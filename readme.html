📄 README — Symbol Open System v2.0 · Aurora Build

Visão Geral

O SOS v2.0 — Aurora Build é a atualização completa do ecossistema Symbol Open System, projetada para unificar design, autogestão e suporte simbiótico em uma interface navegável e modular.

📂 Estrutura do Projeto

  • index.html — Mandala central e navegação geral
  • core.html — Núcleo central (governança, linha do tempo, changelog)
  • design.html — Sistema visual (paleta, tokens e tipografia)
  • support.html — Comandos simbióticos e práticas (Lume, Sorte, ciclo completo)
  • modules/ — Páginas independentes para cada Guardião (Farol, Íris, Joaquina, Lume, Mira, Nexo, Trino) + Corrente 🌬️ Sorte
  • style/main.css — Tema base com tokens visuais e responsividade
  • assets/ — Ícones, imagens e elementos gráficos

🧭 Filosofia v2.0

A versão 2.0 introduz uma lógica de navegação modular: cada Guardião é um “módulo funcional” que reflete uma vertical de autogestão. O sistema assume o papel de espelho e guia, permitindo interação viva entre as camadas simbólicas.

  • Estrutura viva → guardiões como módulos, não apenas conceitos.
  • Design tokenizado → cores e formas derivam da função simbólica.
  • Currents → integração da corrente 🌬️ Sorte como nutriente transversal.
  • Responsividade total → layouts A4, 16:9 e 9:16 já otimizados para expansão.

⚙️ Instalação e Uso

  1. Descompacte o arquivo ZIP em uma pasta local.
  2. Abra o arquivo index.html em seu navegador.
  3. Explore as seções ou módulos via links no cabeçalho.
  4. Para editar, use qualquer editor de texto ou HTML (Visual Studio Code recomendado).
